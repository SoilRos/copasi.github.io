Version 4.17.133

Release Date: November 19, 2016

New Features since COPASI 4.16 (Build 104)

  • Graphical User Interface (CopasiUI)
    • Support for user defined units.
    • Extended unit selection options for time, volume and quantity of the model. These extended options include user defined units.
    • Unit support for Global Quantities.
    • Undo and redo support for all model changes.
    • Improved user interface to method parameters.
    • Selection of matrix and vector elements by name in addition to the index for plots, reports, and in mathematicsl expressions.
    • Support for contour plots.
  • Simulation Engine
    • New hybrid simulation algorithm: (Hybrid RK-45) which supports arbitrary discrete events and user defined ODEs. Additionally, the user can specify which reaction are treated continously and which are treated discretely. This algorithm passes the SBML semantic test suite and stochastic suite.
  • SBML
    • Full support for SBML Level 3.1 event semantics
    • Import and export of unit definitions.
