Version 4.37.264

Release Date: October 5, 2022

New Features since COPASI 4.36 (Build 260)

  • Graphical User Interface (CopasiUI)
    • Import dialogs for COMBINE archives or SED-ML files with importable entities like SBML models, SED-ML files, or SED-ML tasks.
  • Command Line (CopasiSE)
    • Support for handling COMBINE archives or SED-ML files with importable entities like SBML models, SED-ML files, or SED-ML tasks.
  • Installation
      Installation package for Mac OS Big Sur and later on Intel platform
  • Language Bindings
      Java bindings for Mac OSC package on Arm platform
  • Standards
      Enhanced interfacing with COMBINE archives and SED-ML files in CopasiUI and CopasiSE

Bug Fixes since COPASI 4.36 (Build 260)

  • Graphical User Interface (CopasiUI)
    • Improved the layout of optimization and parameter estimation tasks by allocating more space to parameters
    • Removed occasionally occurring spurious error messages when entering the parameter estimation tasks widget.
  • Simulation Engine
    • Improved handling of interdependent parameter ranges in optimization and parameter estimation.
    • Fixed possible problem loading COPASI Files created with COPASI prior to Version 4.2.23
