COPASI 4.7 (Build 34)

These language bindings are based on the stable release COPASI 4.7 (Build 34).

The language binding binaries for Windows have been build using Visual Studio 2010. This means that in order to run the language bindings under windows, the Visual Studio 2010 runtime files have to be installed. If you intend to use the 32 Bit language bindings under windows, you need to install the (32 bit runtime files) and if you intend to run the 64 bit version of the bindings, you need the (64 bit runtime files).

Please see the minimum system requirements for the runtimes, especially with respect to the patch level you need for your version of windows. Further information regarding the language bindings can be found here.

Since we only have limited possibilities for testing different bindings with different operating systems, feedback is always welcome.
