Older Versions of Language Bindings

Starting with this release, I am uploading language bindings for some new languages. This is experimental work and I am not familiar with these languages. Due to this, there is no documentation for them yet, but the API is similar to those of the other language bindings so please have a look there. All nine example files that are distributed with the other language bindings however have been translated to the new languages and are distributed with them.

Right now, we have language bindings for Perl 5 and .NET on MacOS X 10.6, MacOS X 10.7 and Linux which seem to work.

The windows version actually did build, but out of some reason they refuse to run so far. This is probably due to my limited knowledge of these languages on the one hand and windows on the other. I hope to fix these problems as soon as possible so that I can upload these as well.

Since these new language bindings are very experimental, any feedback would be very welcome.
